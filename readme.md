# Spring-boot OpenAPI starter

## Open-API-Starter
The starter module
based on small-rye implementation
provides the yml file on http:/../openapi

## The business logic
- with JAXRS & RestController endpoints AND OpenAPI annotations
- static openapi.yaml
- a custom ModelReader impl
